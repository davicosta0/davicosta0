##  What I Do / Core Competencies

* **Advanced Web Automation & Anti-Bot Evasion:** Building fault-tolerant bots and complex data acquisition systems. I specialize in bypassing advanced anti-bot protections (like Cloudflare) and orchestrating **Headless browser automation** (SeleniumBase, Xvfb) on remote Linux servers.
* **Unstructured Data Structuring (OCR & NLP):** Taming the wildest datasets. I use Computer Vision (PaddleOCR) and Deterministic Logic (Advanced Regex) to extract PII (Personally Identifiable Information) from highly irregular scanned documents with near-zero cloud leaks.
* **Privacy-by-Design Local AI Pipelines:** Architecting hybrid pipelines that combine the semantic intelligence of **Local Open Source LLMs** (running 100% offline via Ollama/Llama 3.x) with deterministic code. This ensures strict PII processing and **GDPR/CCPA compliance**—zero data leaves the infrastructure.
* **High-Performance API Design:** Developing fast, documentable, and scalable RESTful APIs using **FastAPI** to deliver the data sanitized by my automated pipelines.
